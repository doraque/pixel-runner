# Pixel Runner

Tiny endless-runner on a plain HTML `<canvas>`. Pure HTML/CSS/JS, no build-in tools.

## Play
- Open `index.html` in a browser, or enable GitHub Pages on the repo.

## Controls
- ← / → : Move
- Space : Jump
- P : Pause
- R : Restart

## Assets
Place images in `/assets`: `bg.png`, `hero.png` (32×32), `coin.png` (16×16), `spike.png` (24×24).

## Play Online
Deployed via GitHub Pages: https://fenderbrender.github.io/pixel-runner/gamefile
